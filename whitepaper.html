Light-backed, floor-protected digital colours.

Whitepaper v0.2 • 2025 Draft

1. Vision

CoolourCoin introduces a colour-first economy: every token is a unique shade of light, and once that shade receives a price, it can never be sold for less. The goal is to create a calm, transparent place for storing value in a beautiful form.

2. How It Works

Unique colours. Start from a finite Genesis Palette (~4,096 colours). Each colour has an ID, HEX/RGB value and rarity tier.

Semi-random appearance. Colours appear based on a semi-random oscillator seeded by real-time team data, so no one can precompute the whole schedule.

RGB social mining. When a colour becomes available (e.g. R200 G128 B099), 3 different users with R, G, B licences must match it exactly to claim it.

Licences. Licences are paid entry tickets for R/G/B channels. Because they cost money, mined colours inherit real value.

3. The Rule of Light (Spectral Floor)

Once a colour has been bought or mined with paid licences, that price becomes its Spectral Floor. Any future sale of that colour inside the CoolourCoin ecosystem must be at or above that floor.

mapping(uint256 => uint256) public spectralFloor; // colorId => price in wei

// on sale:
// require(msg.value >= spectralFloor[colorId], "Below spectral floor");

No passive yield: CoolourCoin offers no yield, no staking and no airdrops. Its only promise is value preservation.

4. Treasury & Contributors

A public treasury holds rare / near-white colours. These can be used to pay contributors, reward early adopters or be converted into mining licences at the current rate.

5. Launch Path

Phase 1 — website & Genesis list. Phase 2 — palette & licence MVP. Phase 3 — smart contract with spectral floor. Phase 4 — brand onboarding and auctions.

6. Disclaimer

CoolourCoin is an experimental digital-colour protocol. It does not guarantee profit and should be used with audited contracts only.
