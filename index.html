Put your value in light. Let it rest.

A single coin, an entire spectrum. No colour below its own light.

Spectral Floor

Each colour remembers the highest price it ever reached. Smart contracts enforce that it can never be sold below that spectral floor.

RGB Social Mining

Colours are mined only by teams of three: one Red, one Green, one Blue. Mining requires presence, not hardware farms.

No Passive Yield

CoolourCoin offers no yield, staking or “risk-free APR”. It is an instrument for preserving value in light, not inflating it.

Finite Genesis Palette

The Genesis Spectrum holds a finite number of base colours (4 096 blocks). Every colour is born only once and remains traceable forever.

Transparent & Open Source

Protocol code is auditable. Floor logic, mining rules and fees are visible on-chain. Trust is not marketing — it is implementation.

Company-grade Gifting

Colours can be gifted to employees, creators or communities as proof-of-gratitude assets — not as speculative lottery tickets.

Soft Launch, Hard Guarantees

The launch phase stays small and controlled, but the spectral floor rule is strict from day one. No admin can override it.

Military-style Simplicity

One protocol, one rule: never sell below the light it cost. Everything else is optional mechanics built around that core.
