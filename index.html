Protocol at a glance

Finite spectrum

The protocol starts from a fixed, public Genesis Spectrum of 4,096 base colours. No hidden mint. No infinite supply.

Spectral Floor Rule

Each colour remembers the highest price it has ever been bought for. Smart contracts enforce: no sale below that floor.

RGB team mining

New colours are mined only by live Red + Green + Blue teams. No GPUs, no hardware race — just human coordination in real time.

Licence-based entry

Mining access is gated by time-limited licences on R, G and B channels. Licences are the main way value enters the system.

Direct colour purchase

Some colours can be bought directly from the protocol at a deterministic price, always above expected mining cost — keeping miners and buyers aligned.

Transparent treasury

A fraction of licence revenue feeds a transparent Treasury, which can hold colours or pay contributors. All movements are visible on-chain.

No yield, no leverage

CoolourCoin does not pay interest and does not use leverage. It protects floors; it does not promise profit.

Open and auditable

Core contract code is open-source and audit-ready. The main guarantee: you cannot push a colour below its own history.
